Vertex Addons provides a set of **Content & Layout** widgets to help structure your page content and improve design without coding. These widgets let you add advanced content elements like accordions, tabs, info boxes, and more to your Elementor pages. All the widgets are available in the Elementor editor under the **"Vertex Addons"** category (and **"Vertex Addons Pro"** for premium widgets).

<Aside>
Make sure the Vertex Addons plugin (and Pro add-on, if applicable) is installed and activated. You'll then find the widgets listed below in your Elementor sidebar under Vertex Addons sections.
</Aside>

Below is an overview of each Content & Layout widget and what you can achieve with it:

<CardGrid>
  <LinkCard href="./accordion" title="Accordion" description="Collapsible sections to organize content (great for FAQs)."/>
  <LinkCard href="./tabs" title="Tabs" description="Tabbed content areas for switching between different content sections."/>
  <LinkCard href="./toggle" title="Toggle" description="Similar to Accordion but allows multiple sections open at once."/>
  <LinkCard href="./information-box" title="Information Box" description="Icon + text box to highlight information or features."/>
  <LinkCard href="./notice-box" title="Notice Box" description="Stylized notice box with icon and text for announcements."/>
  <LinkCard href="./alert-box" title="Alert Box" description="Attention-grabbing alert box with message and optional close button."/>
  <LinkCard href="./fancy-text" title="Fancy Text" description="Animated text that cycles through words with creative effects."/>
  <LinkCard href="./creative-button" title="Creative Button" description="Buttons with unique hover animations to enhance calls-to-action."/>
  <LinkCard href="./flip-box" title="Flip Box" description="Content box that flips on hover to reveal additional content on the back."/>
  <LinkCard href="./price-box" title="Price Box" description="Pricing table card showing price, features, and a signup button."/>
  <LinkCard href="./slides" title="Slides" description="Image/content slideshow with headings, text, and optional call-to-action."/>
</CardGrid>

Use these widgets to improve your page layout and user engagement. Click on any widget above to see detailed documentation and usage instructions for that element.
