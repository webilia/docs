Installing Vertex Addons is straightforward. You can install the free plugin from the WordPress repository, or upload the ZIP file manually. If you purchase Vertex Pro, install it *in addition* to the free plugin.

## Requirements
- WordPress 5.0 or higher  
- PHP 7.2 or higher  
- Elementor (latest free version)  

## Method 1: Install via WordPress Plugin Directory <Badge text="Recommended" variant="note" />
1. Log in to WordPress admin.  
2. Go to **Plugins → Add New**.  
3. Search for **“Vertex Addons for Elementor”**.  
4. Click **Install Now**, then **Activate**.  

![Vertex in WP Search Results](https://cdn.webilia.com/u/d/2025/08/vertex-in-plugin-search-results.png)

## Method 2: Install via Plugin ZIP Upload 
1. Download the latest Vertex Addons ZIP.  
2. Go to **Plugins → Add New → Upload Plugin**.  
3. Select the ZIP file and click **Install Now**.  
4. Click **Activate Plugin**.  

![Upload Vertex by Zip File](https://cdn.webilia.com/u/d/2025/08/vertex-install-by-upload.png)

<Aside type="caution">If you install Vertex Pro, you will also need to enter your license key under **Vertex Addons → License** in the WP admin.</Aside>
