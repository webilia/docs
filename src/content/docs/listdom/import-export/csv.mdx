## CSV Import & Export

With the **Listdom CSV importer and exporter** (built into the Listdom core), you can easily import classifieds to your listing website with just a few clicks. This tool simplifies the process of extending your online listing and classifieds website by allowing you to import listings in bulk.

After the Listdom CSV add-on is installed and activated, it will add a new **CSV** tab in **Listdom → Import / Export**. This tab has four options:

- Import  
- Mapping Templates  
- Auto Import *(CSV Addon only)*  
- Export  

---

## Import

In this section, you can select your CSV file, map the fields with Listdom default fields, and click **Import** to process.

### Mapping Table

A table to map the Listdom default fields with the columns of the uploaded CSV file. The columns of this table are:

- **Listdom Field:** The default fields of Listdom (e.g. title, price, tags, categories, etc.). Required fields are marked with a red star.  
- **Type:** The type of field (text, number, URL, email, etc.).  
- **Mapping:** The columns of the uploaded CSV file to assign to Listdom fields one by one.  
- **Default Value:** Optional value to insert when the CSV column is empty.  

**Listdom Field Definitions:**

1. **Unique ID** – Required if you want to update listings later.  
2. **Listing Title*** – Required.  
3. **Listing Content** – Description of the listing.  
4. **Listing Date** – Date of listing creation.  
5. **Listing Owner** – If value doesn’t exist, CSV add-on creates it automatically.  
6. **Listing Status** – Published, Draft, Trash, Pending, Expired, On Hold, Offline.  
7. **Price**  
8. **Price Max**  
9. **Price After** – Price description.  
10. **Currency**  
11. **Listing Address**  
12. **Listing Latitude**  
13. **Listing Longitude**  
14. **Listing Link**  
15. **Listing Email**  
16. **Listing Phone**  
17. **Listing Website**  
18. **Listing Contact Address**  
19. **Listing Remark** – Owner message.  
20. **Featured Image** – URL to image.  
21. **Listing Gallery** – Comma-separated image URLs.  
22. **Listing Category** – If new, add-on creates it.  
23. **Listing Locations** – If new, add-on creates it.  
24. **Listing Tags** – If new, add-on creates it.  
25. **Listing Features** – If new, add-on creates it.  
26. **Listing Labels** – If new, add-on creates it.  
27. **Facebook** – Account URL.  
28. **Twitter** – Account URL.  
29. **Instagram** – Account URL.  
30. **WhatsApp** – Number URL.  
31. **YouTube** – Account URL.  
32. **TikTok** – Account URL.  
33. **Telegram** – Account URL.  
34. **Visible Until** – Final visibility date.  

<Aside type="note">
The **Visible Until** field is only available in the Listdom Visibility Add-on.
</Aside>

## How to Add and Import Custom Fields

To import custom fields via CSV:

1. First, create the fields inside Listdom.  
2. Add corresponding columns to your CSV file.  
3. Map them in the import screen.  

### Available Methods

- **Custom Fields (built-in):** Basic fields like text, number, dropdown.  
- **ACF Plugin + Listdom ACF Addon:** Supports advanced fields (gallery, checkbox, color picker, etc.).  

<Aside type="tip">
Fields inside **ACF Accordion and Tabs** will also be imported.
</Aside>

## How to Import Taxonomies

To import taxonomies (categories, locations, tags, features, labels):

1. Create a sample listing with a **Unique ID**.  
2. In your CSV file, add taxonomy terms in the relevant columns (comma-separated).  
3. Import the file — new terms will be created and associated automatically.  
4. Delete the sample listing if desired (terms remain).

## Auto Import (CSV Addon only)

The **Auto Import** option allows scheduled imports from a data source.

<Steps>
1. Enter a **Feed URL** – public CSV link (e.g. from Google Sheets).  
2. Select a **Field Mapping Template**.  
3. Choose an **Interval** – how often to fetch.  
4. Define **Import Size** – maximum number of listings per run.  
</Steps>

**Google Sheets Example:**

https://docs.google.com/spreadsheets/d/FILE_ID/export?format=csv&id=FILE_ID&gid=SHEET_GID

<Aside type="note">
Listdom will fetch the latest data from this URL at the defined interval and update listings automatically.
</Aside>
