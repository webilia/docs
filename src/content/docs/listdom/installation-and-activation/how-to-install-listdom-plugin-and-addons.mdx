To get started, you'll need to install the **Listdom Classification WordPress plugin**, which offers several powerful addons.  
Here's a step-by-step guide to help you install each component and unlock the full potential of your site.

The **Listdom plugin** is free and also has several free and paid addons and themes.  
You can download Listdom from the WordPress plugins repository and buy the addons/themes from **Webilia**. After purchase, you can download the installation files from the **Downloads tab** in your Webilia account.

## Upload via FTP
<Steps>
1. Unzip the `.zip` file that you've downloaded.  
2. Upload the extracted folder into `/wp-content/plugins/`.  
3. In your WordPress dashboard, go to **Plugins → Installed Plugins**.  
4. Activate the installed plugin.
</Steps>

## Upload via WordPress Dashboard
<Steps>
1. Navigate to **Plugins → Installed Plugins**.  
2. Click **Add New** and then **Upload Plugin**.  
3. Click **Choose File**, select the downloaded ZIP file, and hit **Install Now**.  
4. Once installed, go to **Plugins → Installed Plugins** and activate it.
</Steps>

## License Activation
Now the final step is to enter the **license code** and activate any paid Listdom add-ons with a license key.  
For details, read the full guide on [how to activate the Listdom products](#).

<Aside type="tip">
Keep your license active to access plugin updates and official Webilia support.
</Aside>
