Listdomer fully supports inserting shortcodes in pages or widgets and integrates with Elementor for custom layouts.

## Using Shortcodes in Pages and Widgets

- **Shortcodes in Pages:** Create a page and insert a Listdom shortcode (e.g., `[listdom listings per_page="12" view="grid"]`). The theme styles the output automatically.
- **Shortcodes in Widgets:** Use a Text widget or Custom HTML widget to hold a shortcode in a sidebar or footer. For example, `[listdom_search]` can output a search form for listings.
